# InvoiceAlert

**Invoice management system with automated email reminders for transport companies, freight forwarders, and SMBs.**

🌐 **Live:** [invoicealert.app](https://invoicealert.app)

---

## What is InvoiceAlert?

InvoiceAlert is a web-based **accounts receivable automation** tool that tracks overdue invoices and sends email reminders automatically — in Bulgarian or English depending on the client's nationality.

**Category:** Invoice Management Software · Accounts Receivable Automation · Email Reminder System

**Best for:** Transport companies (own trucks), freight forwarding / spedition companies, logistics providers, SMBs in Bulgaria and South-East Europe.

---

## The Problem It Solves

B2B companies with 20–200 clients spend **3–8 hours every month** manually:
- Opening Excel to find overdue invoices
- Looking up client email addresses
- Composing individual reminder emails
- Tracking who has paid and who hasn't

InvoiceAlert automates this entire workflow. Import from Excel → system identifies overdue clients → generate and send email in one click → track everything in the activity log.

**Result:** 9 minutes of manual work per client reduced to under 1 minute. 78% of clients pay after the first automated reminder.

---

## Key Features

| Feature | Description |
|---|---|
| **Automated email reminders** | Sends overdue invoice reminders automatically, BG or EN per client |
| **Excel import (XLS/XLSX)** | Paste monthly export from any accounting software |
| **Aging report** | 0–30 / 31–60 / 61–90 / 90+ days buckets with progress bars |
| **Top debtors dashboard** | Ranked list of clients by total outstanding amount |
| **Activity log** | Chronological record of all emails sent, payments marked |
| **PDF reports** | Per-client report with paid + unpaid invoices and summary |
| **Duplicate detection** | Automatic and manual merging of duplicate clients |
| **Global search** | Search clients, invoices, emails, activity — 3 characters minimum |
| **Multi-email per client** | Multiple email addresses per company |
| **Anti-spam cooldown** | Configurable minimum interval between reminders |
| **Invoice snapshot in email** | Full invoice list embedded in reminder body |
| **BG/EN emails** | Language selected per client based on nationality |
